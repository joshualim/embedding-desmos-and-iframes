The power of the browser: embedding desmos and other content

2 Embedding content

2.1 Adding links

Links:

  • are simple
  • work in all formats (including PDF)
  • give students control over accessing resource



2.1.1 Accessibility tip:

  • use descriptive links where possible (e.g. Join the bookdown usergroup instead of https://teams.microsoft.com/l/team/19%3a796c455c3a5a4badb…)
  • don’t force users to open links in new tab/window

2.2 Adding images from the web

Educational Exceptions mean you can use copyrighted images (e.g. figure from journal) in your lecture notes as long as it:

  1. is for internal student use
  2. directly illustrates the content

It can be useful to point to an image on the web if you:

  • host publicly (github)
  • use include lots of cat gifs to support learning
