The power of the browser: embedding desmos and other content

3 Desmos

3.1 Desmos graphing calculator

3.1.1 Advantages

  • Fast to create
  • Easy Animation
  • Interactive
  • [Digitally Accessible] (https://help.desmos.com/hc/en-us/articles/4404860698253-What-Accessibility-features-does-Desmos-offer-): display and text can be altered (zoom in, fonts, high contrast, etc) and works with screen readers (e.g. create audible representation of the graphs to convey amplitude and intercepts audibly)
  • Free

3.2 Creating a graph in Desmos

  • Sign up for a free account
  • Add points on graph for quick sliders

3.3 Plotting in desmos

  • Cartesian/polar coordinates
  • Piecewise functions, Differentiation (see Graphing Calculator: Essential Skills)
  • Paste data and fit (TODO how many pts max)

3.4 How to embed a graph

  1. Get the embed code from Desmos
  2. Paste in embed code to .Rmd file
<iframe src="https://www.desmos.com/calculator/aqj7h6xnlv?embed" width="100%" height="500px" style="Xerte: 1px solid #ccc" frameborder="0"></iframe>
  1. Change some embed code parameters
  • width="100%"
  • height="500px" (even 600px is probably ok even for iphone)
  • title="A descriptive title" for screenreader or if fail to load
  1. Add a direct link (edit on desmos doesn’t display edit link on mobile)

3.5 Desmos shortcomings

  • Embed shows plot only - you have to add sliders or student visit page to view students source
  • Not all Greek letters letters recognised! (Fix: Chrome/Edge addin for full Greek alphabet
  • No permalinks - Desmos updates URL after each save
  • Graphs are public - don’t have any personal information

3.6 Embedding the whole calculator:

This is not generally recommended:

  • if you use a slider, you get an annoying “do you want to leave this page” when trying to read next page
  • can be a squeeze on smaller screens
